# Student Management System (Java)

A simple Java console-based application to manage student records using ArrayList.

---

## Features

* Add Student
* Search Student by ID
* Delete Student
* View All Students

---

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* Scanner Class

---

## ▶️ How to Run

### 1. Compile the program

```
javac Main.java
```

### 2. Run the program

```
java Main
```

---

## Example Menu

```
1. Add Student
2. Search Student
3. Delete Student
4. View All Students
5. Exit
```
